# EduBot - Education Chatbot

A domain-restricted AI chatbot for education topics with face verification.

## Features
- Face ID verification (multi-user)
- Education domain filtering
- Chat history per user
- Built with Flask + Groq (Llama 3.3)

## Run Locally
```
pip install -r requirements.txt
python app.py
```

## Deploy on Render
1. Push to GitHub
2. Connect repo on render.com
3. Set environment variable: GROQ_API_KEY
4. Deploy!
